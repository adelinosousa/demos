# License Audit

Check .net project nuget packages licenses

## Usage

```bash
License.Audit -p C:\Path\To\NetFramework\Project
```

### Options

| Parameter | Description                         |
| --------- | ----------------------------------- |
| -p        | Path to .net project root directory |
| -o        | Outputs result to a csv file        |
